WaxOn

Raw Recording Prep

Use WaxOn on raw recordings before you edit. Drop your files in, configure what you care about, and get to work.

High-Pass Filter

Configurable cutoff from 20 to 90 Hz. Removes rumble, HVAC hum, and handling noise. Set to 20 Hz for DC block only.

Noise Reduction

Optional RNNoise ML noise suppression — a recurrent neural network trained on speech. Reduces steady background noise before any other processing. Off by default.

Loudness Normalization

Optional two-pass EBU R128 with configurable target (−35 to −16 LUFS). Linear gain only — dynamics fully preserved.

Brick-Wall Limiting

2× oversampled true peak control at your chosen ceiling (−1 to −3 dB). Keeps transients honest.

Phase Rotation

200 Hz allpass reduces peak asymmetry and improves limiter headroom on voice recordings.

Mix Mode

Select 2+ files and combine them. When loudness norm is on, each file is individually leveled before mixing, then the combined output is normalized — ensuring a balanced blend.

Presets & Batch Processing

Five built-in presets plus custom presets. Up to 3 concurrent jobs with per-file progress.

Output: {name}-{rate}waxon-{limit}.wav  ·  24-bit WAV

WaxOff

Delivery & Mastering

Run your finished, edited mix through WaxOff. Broadcast-standard loudness normalization, true peak control, and MP3 output.

EBU R128 Normalization

Two-pass analysis plus linear gain. No dynamic processing — stereo image and transients are completely unchanged.

True Peak Control

Configurable ceiling from −3.0 to −0.1 dBTP (default −1.0). Compliant with podcast platform requirements.

WAV + MP3 Output

24-bit WAV, CBR MP3 at 128, 160, or 192 kbps, or both simultaneously. Deliver exactly what your platform needs.

Phase Rotation

Optional 150 Hz allpass to reduce crest factor on bass-heavy material before final delivery.

Output: {name}-lev-{target}LUFS.wav / .mp3

Workflow

The Full Picture

WaxOn and WaxOff are two stages of the same pipeline — raw prep before editing, delivery prep after.

Raw Recordings → WaxOn → Edit in DAW → WaxOff → Distribute

Supported formats: WAV, AIFF, AIF, MP3, FLAC, M4A, OGG, Opus, CAF, WMA, AAC, MP4, MOV. FFmpeg is bundled — no separate installation required.
